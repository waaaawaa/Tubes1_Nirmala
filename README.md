# Tubes Robocode Greedy

## Deskripsi
Program ini merupakan implementasi algoritma greedy pada permainan Robocode Tank Royale menggunakan bahasa C#. Bot dirancang untuk dapat bergerak, mendeteksi musuh, menyerang, dan bertahan secara otomatis berdasarkan kondisi permainan secara real-time. :contentReference[oaicite:0]{index=0}

## Algoritma Greedy
Bot menggunakan strategi greedy dengan memilih tindakan terbaik pada setiap kondisi permainan.

- BOTNIRMALA → menyerang musuh terdekat secara agresif
- BOTKEDUA → fokus menyerang terus menerus
- BOTKETIGA → fokus bertahan dan menghindari serangan
- BOTKEEMPAT → patroli arena sambil scanning musuh :contentReference[oaicite:1]{index=1}

## Requirement
Program membutuhkan:
- Java JDK
- Robocode Tank Royale
- Visual Studio Code / Visual Studio
- Bahasa pemrograman C# :contentReference[oaicite:2]{index=2}

## Cara Menjalankan
1. Clone repository GitHub
2. Jalankan Robocode Tank Royale
3. Masukkan file bot ke project
4. Compile program bot
5. Jalankan pertandingan pada Robocode Tank Royale

## Kendala Development
Beberapa kendala yang dihadapi saat pengembangan:
- Bot terkadang menabrak dinding arena
- Radar kehilangan target musuh
- Bot masih terlalu fokus menyerang
- Strategi greedy hanya mempertimbangkan kondisi saat ini :contentReference[oaicite:3]{index=3}
